# Module 1 Study Guide

## Getting Started & Home Tab Basics

### Key Topics
- **File Tab**: Understanding the Backstage view, saving documents, and opening files.
- **Clipboard**: Using Cut, Copy, Paste, and Format Painter effectively.
- **Font**: Changing font type, size, color, and applying basic effects (Bold, Italic, Underline).

### Quick Tips
- Use `Ctrl + S` to save your work frequently.
- Use `Ctrl + C` and `Ctrl + V` for quick copying and pasting.
- The Format Painter allows you to copy formatting from one text selection to another.

# Module 2 Study Guide

## Home Tab Formatting & Styles

### Key Topics
- **Paragraph**: Adjusting alignment, indentation, line spacing, and lists (bullets/numbering).
- **Styles**: Applying and modifying built-in styles for consistent headings and text.
- **Editing**: Using Find, Replace, and Select tools to navigate large documents.

### Quick Tips
- Use Styles instead of manual formatting for easier updates later.
- Use `Ctrl + E` for center alignment and `Ctrl + L` for left alignment.
- The Navigation Pane helps you jump between headings quickly.

---

# Module 3 Study Guide

## Insert Tab Features

### Key Topics
- **Tables**: Creating, formatting, and managing data within tables.
- **Images & Shapes**: Inserting pictures, icons, and shapes with text wrapping.
- **Headers & Footers**: Adding consistent information to every page.
- **Page Numbers**: Inserting and formatting page numbers.

### Quick Tips
- Use "Wrap Text" options to control how images interact with text.
- Headers and Footers are ideal for document titles, dates, and page numbers.
- Tables can be styled using the Table Design tab for professional looks.

---

# Module 4 Study Guide

## Document Design & Layout

### Key Topics
- **Design Tab**: Applying themes, colors, and fonts to change the document's overall look.
- **Page Layout**: Setting margins, orientation (Portrait/Landscape), and paper size.
- **References**: Creating a Table of Contents, adding citations, and managing bibliographies.

### Quick Tips
- Always set your margins and paper size before starting heavy formatting.
- Use automatic Table of Contents based on Styles (Heading 1, Heading 2, etc.).
- Themes ensure all colors and fonts match throughout the document.

---

# Module 5 Study Guide

## Review & Finalization

### Key Topics
- **Review Tab**: Using Spelling & Grammar check, Thesaurus, and Word Count.
- **View Tab**: Switching between Print Layout, Web Layout, and Reading View.
- **Finalizing**: Protecting documents, checking accessibility, and exporting to PDF.

### Quick Tips
- Always run Spelling & Grammar check before sharing a document.
- Use "Read Mode" for distraction-free reading.
- Exporting to PDF ensures your formatting stays fixed on other devices.
